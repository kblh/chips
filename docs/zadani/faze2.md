# Fáze 2

web bude mít kromě informativní složky také tyto funkcionality:

- formulář na přihlášení talku
- program aktuáního eventu
- archiv proběhlých eventů (blog)
- hlasování pomocí QR kódu během eventu pro nejlepší talk

Cílem je držet web technologicky co nejjednoduší. Co nejjednoduší stack, nejlépe bez DB, bez BE.
Nejlépe bych chtěl využít již existujících služeb (Google Forms, nějaké free survey aplikace) a propojit vše pomocí API. Uděleh hlubší průzkum a dej mi na výběr z více variant.
Nejprve potřebuji vymyslet cestu, kterou se vydat.
